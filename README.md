# Artes All Stars Party



- Spelet i detta läge har en gamemode man kan köra från början till slut. Den andra gamemoden är för tillfället "borttaget" från spelet p.g.a att jag håller på med överföring av 2D logik och assets till 3D.

Instruktioner till Kim:

- När du öppnat projektet kan du bekanta dig med scenerna och scenernas innehåll. 2 är testscener (Environment test & VFX test), Rhythmic Gymnastics och Testing Environment är minigame scener (Testing Environment är Artes Fight Club och inte för tillfället i bruk). Sedan finns MainMenu som är en 2D scen.
- Du kan antingen köra spelet genom att köra MainMenu scenen ELLER så går du till File -> Build and Run
- Dina controller är ASWD, musens left click, spacebar & pilarna. En ordentligare förklaring på kontrollerna kommer senare i spelet också.
